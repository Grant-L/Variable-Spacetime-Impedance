\label{ch:universal_operators}

The complete derivation chain spans 14 orders of magnitude
in length scale (from the lattice pitch $\ell_{node} \approx 3.86 \times
10^{-13}$~m to the Hubble radius $R_H \approx 1.33 \times 10^{26}$~m).
A persistent question is whether these scales are merely analogical---linked
by suggestive but structurally unrelated mathematics---or whether the
\textit{same operator} acts at every scale.

This book proves the latter.

\section{The Universal Saturation Operator}
\label{sec:universal_saturation}

The saturation factor
\begin{resultbox}{Universal Saturation Factor ($S$)}
\begin{equation}
    S(A, A_c) = \sqrt{1 - (A/A_c)^2}
    \label{eq:saturation_sigma}
\end{equation}
\end{resultbox}
appears in four structurally distinct domains:

\begin{enumerate}
    \item \textbf{Dielectric saturation} (Axiom 4): lattice permittivity
          collapses under strong-field loading.  In fluid-mechanics language,
          this is the Bingham plastic yield: the vacuum flows above
          $\tau_y = B_\text{snap}^2/2\mu_0$.
    \item \textbf{BCS critical field}: $B_c(T) = B_{c0}\sqrt{1-(T/T_c)^2}$.
    \item \textbf{Galactic rotation}: $\eta_{eff} = \eta_0\sqrt{1-(g_N/a_0)^2}$.
    \item \textbf{Relativistic mass--energy}: $E = mc^2/\sqrt{1-(v/c)^2}$.
\end{enumerate}

All four call \textit{the same engine function}
(\texttt{scale\_invariant.saturation\_factor(A, A\_yield)}) that serves
all four applications.

\section{The Universal Pairwise Potential}
\label{sec:universal_pairwise}

The pairwise interaction between two nodes at separation $r$ is constructed
by composing Operators~1--3:

\begin{resultbox}{Universal Pairwise Potential ($U$)}
\begin{equation}
    U(r) = -\frac{K}{r}\bigl(T^2 - \Gamma^2\bigr), \qquad
    \Gamma(r) = \frac{Z(r) - Z_0}{Z(r) + Z_0}, \qquad
    Z(r) = \frac{Z_0}{\bigl(1 - (d_\text{sat}/r)^2\bigr)^{1/4}}

\end{resultbox}

\section{The Universal Eigenvalue Target}
\label{sec:universal_eigenvalue}

The ground state of any impedance network is the configuration where
at least one mode is perfectly absorbed:
\begin{resultbox}{Eigenvalue Target}
\begin{equation}
    \lambda_\text{min}\bigl(S^\dagger S\bigr) \to 0
    \label{eq:eigenvalue_target}
\end{equation}
\end{resultbox}

When $\lambda_\text{min} = 0$, the network has found an eigenstate---a
zero singular value of $[S]$ means one mode passes through the system
with no reflection.  Code path: \texttt{universal\_operators.universal\_eigenvalue\_target(S)}.

\section{The Universal Packing Reflection}
\label{sec:universal_packing}

Axiom~3 applied at the macroscopic scale: any confined system of $N$
nodes has an equilibrium packing fraction $\eta = P_C(1 - 1/N)$ from
Axiom~4.  The macroscopic reflection coefficient measures spatial
mismatch:
\begin{resultbox}{Packing Reflection Coefficient ($\Gamma_\text{pack}$)}
\begin{align}
    R_{g,\text{target}} &= \sqrt{\tfrac{3}{5}}\left(
      \frac{3\,N\,V_\text{res}}{4\pi\,\eta_\text{target}}
    \right)^{1/3}, \qquad
    \eta_\text{target} = P_C\!\left(1 - \frac{1}{N}\right)
    \label{eq:rg_target} \\[4pt]
    \Gamma_\text{pack} &= \frac{R_g - R_{g,\text{target}}}
                              {R_g + R_{g,\text{target}}}
    \label{eq:gamma_pack}
\end{align}
\end{resultbox}

This operator is domain-agnostic: it applies to protein globules,
nuclear matter ($r_\text{node} = r_\text{proton}$), and any fluid in a
finite cavity.

Code path: \texttt{universal\_operators.universal\_packing\_reflection(Rg\_sq, N, r\_node, P\_C)}.
