# Documentation

This directory contains documentation for the Variable Spacetime Impedance project.

## Structure

- `theory/` - Theoretical derivations and mathematical formulations
- `notes/` - Research notes and ideas
- `papers/` - References to relevant papers and literature
- `figures/` - Generated figures and plots

## Writing Documentation

Documentation can be written in:
- Markdown (`.md`) files
- Jupyter notebooks (`.ipynb`) converted to documentation using `myst-nb`
- LaTeX (`.tex`) for mathematical derivations
